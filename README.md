# Habit Spark

A tiny habit tracker that helps you build streaks with minimal friction.

## Features
- Minimal TypeScript + Node scaffold
- Local JSON persistence (planned)
- Habit completion + streak logic (planned)
- Smoke test to verify scaffold health

## Stack
- Node.js 20+
- TypeScript 5
- npm
- Docker

## Setup
```bash
npm install
```

## Run
```bash
npm run dev
```

## Test
```bash
npm test
```

## Build
```bash
npm run build
npm start
```

## Docker
```bash
docker build -t habit-spark .
docker run --rm habit-spark
```
